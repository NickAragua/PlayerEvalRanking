Usage instructions:

Drag and drop exported registered player list into the bottom right box. This will get you a list of all registered players and their grade levels.
Drag and drop individual coach evals into top right box. You will be able to preview the loaded data there.
Click 'merge eval' to add the current contents of the top right box to the coalesced player list in the top left box. This is used for "current" season evaluations.
Click 'merge previous season eval' to add the current contents of the top right box to the coalesced player list in the top left box.  This is used for "previous" season evaluations.

If you mess up and want to undo your last "merge", click the "Undo" button.
